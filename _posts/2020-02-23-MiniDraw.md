This is a simple drawing app written in C++ based on Qt.

[TOC]

## 1. Feature

* Drawing line, ellipse, rectangle, polygon and other primitives
* Each primitive needs to be encapsulated by a class (object), such as `CLine`, `CEllipse`, `CRect`, `CPolygon`, `CFreehand`
* Setting color and width of the brush
* Color fill
* Draw smooth curve
* Undo
* Save
* Icon

## 2. Development Environment

* **IDE**: Microsoft Visual Studio 2019 community
* **Build**: CMake 3.16.3
* **GUI Platform**: Qt 5.14.1
