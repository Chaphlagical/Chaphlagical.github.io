# 一、巴特沃斯低通滤波器  

巴特沃斯低通滤波器表示为：  
$$H(u,v)=\frac{1}{1+(\frac{D(u,v)}{d})^{2n}}$$  

其中 $$D(u,v)=[(u-P/2)^2+(v-Q/2)^2]^\frac{1}{2}$$  
P和Q为图像填充后的长宽，D(u,v）即表示点(u,v)到中心的距离,n表示阶数。

# 三、巴特沃斯带阻滤波器  

高斯带阻滤波器表示为：  
$$H(u,v)=1-e^{-(\frac{D^2-D_0^2}{DW})^2}$$  

其中 $$D(u,v)=[(u-P/2)^2+(v-Q/2)^2]^\frac{1}{2}$$
